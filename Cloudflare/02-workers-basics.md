# Cloudflare Workers Basics

## 🧠 Concept

Cloudflare Workers allow JavaScript execution at the edge.

---

## ⚙️ Flow

```
Request → Edge Worker → Modify/Route → Response
```

---

## 🔥 Use Cases

- API routing
- Authentication
- Request filtering
- Geo-based routing

---

## 🧠 Telecom Mapping

Workers behave like:
- Policy function (PCRF-like logic)
- Decision engine at edge
